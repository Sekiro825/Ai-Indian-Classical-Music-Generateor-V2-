# AI Indian Classical Music Generator V2

Research and engineering codebase for controllable Indian classical music generation.

## Project Focus

This repository contains two major model lines:
- Hybrid CVAE model family
- V2 Transformer-Flow model family

Both are designed for conditioned generation using controls such as raga, taal, mood, tempo, and duration.
